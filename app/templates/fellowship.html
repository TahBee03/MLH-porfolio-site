Introduction

Welcome to my portfolio! My name is Talike (pronounced tah-leek) and I am a Production Engineering Fellow (Pod 22.SUM.15) in the MLH Fellowship for summer 2022. This website serves as the home of all the information I have soaked in while taking part in the fellowship. Each week focused on a different concept and with every concept, this website continued to grow. Feel free to look around, get an idea of what was covered during the fellowship, and learn a little bit more about me!

Week 1: Orientation

During this week, we learned about the overall flow of the fellowship, got an introduction to production engineering, and established the foundation of our portfolio websites.


Concepts Learned

  • production engineering fundamentals
  • development environments
  • packages + package managers
  • pair coding + code review

Tools Used

  • Python (.py) + Flask + Jinja
  • HTML (.html)
  • CSS (.css)
  • Git + GitHub

Week 2: Linux Fundamentals

During this week, we were introduced to Linux, evaluated its importance to production engineers, and set up our virtual private servers.


Concepts Learned

  • giving and receiving feedback
  • Linux commands + filesystem + terminal
  • virtual private server (VPS)
  • SSH authentication
  • website domain
  • CTCI: Arrays

Tools Used

  • DigitalOcean
  • DuckDNS
  • tmux

Week 3: Scripting

During this week, we went in depth with Bash scripting, learned about processes and signals, and created a script to automate the deployment of our websites.


Concepts Learned

  • scripting
  • deployment automation
  • programs + processes + signals
  • CTCI: String Manipulation + Regex

Tools Used

  • Bash (.sh)

Week 4: Services + Databases

During this week, we were introduced to databases and HTTP requests, and we combined these two concepts in order to establish a database for our websites that we can interact with.


Concepts Learned
